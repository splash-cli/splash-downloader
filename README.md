# splash-downloader
> Download unsplash photos

## Install
```sh
	$ npm install @splash-cli/splash-downloader --save

	#or

	$ yarn add @splash-cli/splash-downloader
```

## Usage
```js
	import download from '@splash-cli/splash-downloader';

	download(flags, photInfo, setAsWallpaper);
	//=> promise
```
