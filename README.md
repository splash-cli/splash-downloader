# splash-download
> Download unsplash photos

## Install
```sh
	$ npm install @splash-cli/splash-download --save

	#or

	$ yarn add @splash-cli/splash-download
```

## Usage
```js
	import download from '@splash-cli/splash-download';

	download(flags, photInfo, setAsWallpaper);
	//=> promise
```
